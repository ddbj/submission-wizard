# Submission Wizard

Demo: https://ursm-submission-wizard.pages.dev/

## Requirements

- Node.js 16.x

## Setup

```
$ npm install --global yarn
$ yarn install
```

## Starting the development server

```
$ yarn start --open
```

## Running tests and linting code

```
$ yarn check-all
```

## Building for production

```
$ yarn build
```
