# Submission Wizard

Demo: https://ursm-submission-wizard.pages.dev/

## Requirements

- Node.js 16.x

## Setup

```
$ npm install --global yarn
$ yarn install
```

## Starting the development server

```
$ yarn start
or
$ yarn start --open
```

## Running tests

```
$ yarn test
or
$ yarn test --watch
```

## Linting

```
$ yarn lint
```

## Building for production

```
$ yarn build
```
